# Introduction

Uncreated.Warfare is an advanced framework for military RP servers made for Uncreated Network. It has a versatile gamemode system designed to allow for community modding.

It implements a full plugin framework from scratch, including commands, events, plugins, entity framework, and helpful abstractions.

It also includes sub-modules for kits, FOBs, death messages, and more.

Gamemodes are created using external DLLs placed in the Plugins folder, and are accompanied by a layout file.
